# IT Infrastructure Management System

## Description
This is a mini project developed to demonstrate IT Infrastructure Management concepts. 
The system provides modules for Service Catalog, Availability Management, Information Security, 
Capacity Management, and Continuity Management.

## Modules Included
- Service Catalog Management
- Availability Management
- Information Security Management
- Capacity Management
- Continuity Management

## Technologies Used
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Tools: Visual Studio Code, GitHub

## Project Structure
backend/
frontend/


## How to Run the Project

### Backend
1. Open terminal in backend folder
2. Run the following commands:
   npm install
node server.js

### Frontend
1. Open terminal in frontend folder
2. Run the following commands:
npm install
npm start


3. Open browser and go to:
   http://localhost:3000/

   
## Conclusion
This project demonstrates a basic IT Infrastructure Management System and shows how software 
can be used to manage IT infrastructure services effectively.
